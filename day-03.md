# Day-03: Git Branch

- git branch - To check your current branch. If there's star near your branch name, then that's your current working branch.
- git branch (your_branch_name) - Creates a new branch
- git checkout (branch_name) - Moves to the requested branch
- git checkout -b (your_branch_name) - Creates and moves to the newly created branch
- git branch -d (your_branch_name) - To delete a branch. (To delete a branch, you have to checkout to the main branch before you delete).
