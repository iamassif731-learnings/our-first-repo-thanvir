- git init - To initialize a git workspace
- git add (file_name or .) - stages a file or entire unstagged files and folders.
- git commit -m "Your-Message" - move the stagged file to commit stage with your personal notes
- git branch -M main - changes name from "master" branch to "main" branch
- git remote add origin (https://your-repo-link) - sets your current local workspace to cloud repo in GitHub. Simply, it connects your local workspace to cloud repo. 
- git push -u origin main - it pushes your local committed files and folders to cloud repo in GitHub main branch. 
- git status - To check current status of git tracker.
